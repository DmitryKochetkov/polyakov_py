# Решения задач Полякова на Python

Структура проекта:

- `Solutions_{}.ipynb` - Jupyter-ноутбук с решением всех задач данного типа
- `answers.csv` - Ответы ко всем задачам, скачанные с сайта Полякова
- `data` - Входные данные к задачам.
